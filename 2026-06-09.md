1. LVM(Logical Volume Manager)
	1.1 파티션(LVM) PV생성 VG생성  LV생성 파일시스템 마운트
	
	`1.2 PV 생성`
```bash
pvcreate /dev/sdb1 /dev/sdb2
pvremove /dev/sdb1 /dev/sdb2
pvs
pvdisplay
```

	1.3 VG 생성
```bash
vgcreate 그룹명 pv명1 pv명2 ...
vgremove 그룹명
vgextend 그룹명 pv명1 ....
vgreduce 그룹명 pv명 .....
vgs
vgdisplay
```

	1.4 LV 생성
```bash
lvcreate 용량 -n vg명 lv명
lvremove /dev/vg명/lv명
lvextend +3G /dev/vg명/lv명 # 기존 용량에 3G추가
lvextend 10G /dev/vg명/lv명 # 최종 용량을 10G로 생성
lvreduce -3G /dev/vg명/lv명 # 데이터 깨지므로 지양
```

```bash
pvs

vgcreate sgm /dev/sdb1
# vgcreate sgm -s 2M /dev/sdb1 # PE Size 2.00 Mib

lvcreate -L 3.3G -n sgm1 sgm
lvcreate -L 3.3G -n sgm2 sgm
vgdisplay #vg 남은 용량 확인 
lvcreate -L 3.3G -n sgm3 sgm
# lvcreate -l 1689 -n sgm1 sgm
lvscan
lvs
lvdisplay

mkfs.ext3 /dev/sgm/sgm1
mkfs.ext4 /dev/sgm/sgm2
mkfs.xfs /dev/sgm/sgm3

mkdir /f1 /f2 /f3

mount /dev/sgm/sgm1 /f1
mount /dev/sgm/sgm2 /f2
mount /dev/sgm/sgm3 /f3

# 확인
lsblk
df -Th
mount
```
