## 그룹정책










**관리 템플릿** 항목을 확인
```
GPO 편집 → 컴퓨터 구성 또는 사용자 구성
→ 정책 → 관리 템플릿
```

중앙 저장소 적용 **전:**
```
관리 템플릿: 로컬 컴퓨터의 정책 정의(ADMX 파일)를 검색했습니다.
```

중앙 저장소 적용 **후:**
```
관리 템플릿: 중앙 저장소에서 정책 정의를 검색했습니다.
```
