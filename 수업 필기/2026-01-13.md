# DNS 서버 구축
___

### 패키지(BIND) 설치
___
```bash
sudo dnf -y update #시스템에 설치된 모든 패키지를 최신 버전으로 업데이트
sudo dnf -y install bind bind-utils #DNS 서버(BIND)와 관련 유틸리티 설치
sudo systemctl enable --now named #DNS 서버 서비스(`named`)를 부팅 시 자동 시작, 즉시 실행
sudo systemctl status named --no-pager #DNS 서버(`named`)의 현재 상태 확인
```
### DNS 설정
___
```bash
sudo cp -a /etc/named.conf /etc/named.conf.bak.$(date +%F) #-a 권한, 소유자, 시간 정보, 심볼릭 링크 등을 모두 유지한 채 재귀적으로 복사 

sudo vi /etc/named.conf #Bind DNS 서버의 메인 설정 파일
```

```bash
options {
        listen-on port 53 { any; }; #any; 로 수정
        listen-on-v6 port 53 { ::1; };
        directory       "/var/named";
        dump-file       "/var/named/data/cache_dump.db";
        statistics-file "/var/named/data/named_stats.txt";
        memstatistics-file "/var/named/data/named_mem_stats.txt";
        secroots-file   "/var/named/data/named.secroots";
        recursing-file  "/var/named/data/named.recursing";
        allow-query     { any; }; #any; 로 수정
        allow-recursion { 192.168.200.0/24; 127.0.0.1; };
        
        dnssec-validation no; #yes 에서 no로 수정
```
