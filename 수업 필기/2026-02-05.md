trunk 다수의 가상랜 정보 전달
access 하나의 가상랜 정보 전달

Sw-net2(config)#vlan 5

Sw-net2(config-if)#switch

Sw-net2(config-if)#switchport mode ?
access Set trunking mode to ACCESS unconditionally
dynamic Set trunking mode to dynamically negotiate access or trunk mode

trunk Set trunking mode to TRUNK unconditionally
Sw-net2(config-if)#switchport mode acc
Sw-net2(config-if)#switchport mode access
Sw-net2(config-if)#swi
Sw-net2(config-if)#switchport acc
Sw-net2(config-if)#switchport access vlan 5
```
port는 기본으로 access
mode는 기본으로 dynamic
상대방이 trunk면 나도 trunk
```
