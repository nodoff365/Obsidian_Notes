| ---------------- | --------------- | ----------------------- |
| idm.kcci.edu     | 192.168.200.120 | IdM 서버 (통합 DNS)         |
| web.kcci.edu     | 192.168.200.121 | 프론트엔드 NGINX 웹 서버/WAF 서버 |
| was.kcci.edu     | 192.168.200.122 | Tomcat WAS서버            |
| mysql.kcci.edu   | 192.168.200.123 | 백엔드 MySQL 데이터베이스 서버     |
| bastion.kcci.edu | 192.168.200.130 | Bastion 서버              |
| dns              | 168.126.63.1    |                         |

```bash
--- VMware 복제 후 웹서버 환경 설정 ---

sudo rm -f /etc/machine-id
sudo systemd-machine-id-setup

sudo hostnamectl set-hostname [호스트명]

sudo vi /etc/hosts
192.168.200.121 web.kcci.edu web

nmcli connection show
sudo nmcli connection modify ens160 ipv4.addresses 192.168.200.121/24
sudo nmcli connection modify ens160 ipv4.dns "192.168.200.120 168.126.63.1"
sudo nmcli connection modify ens160 ipv4.method manual
sudo nmcli connection up ens160
```

## 1단계: IdM 서버 설치 및 구성
### 1.1 IdM 서버 패키지 설치
idm.kcci.edu (192.168.200.120)에서 수행

```bash
# IdM 서버 및 DNS 패키지 설치
sudo dnf install ipa-server ipa-server-dns -y
```

### 1.2 방화벽 규칙 추가
```bash
sudo firewall-cmd --permanent --add-service={http,https,dns,ntp,freeipa-ldap,freeipa-ldaps}
sudo firewall-cmd --permanent --add-port={80/tcp,443/tcp,389/tcp,636/tcp,88/tcp,88/udp,464/tcp,464/udp,53/tcp,53/udp,123/udp}
sudo firewall-cmd --reload
```

### 1.3 IdM 서버 설치 실행
```bash
sudo ipa-server-install \
  --realm KCCI.EDU \
  --domain kcci.edu \
  --ds-password 'Tj0tlf.0505' \
  --admin-password 'Tj0tlf.0505' \
  --hostname idm.kcci.edu \
  --ip-address=192.168.200.120 \
  --setup-dns \
  --forwarder=168.126.63.1 \
  --forwarder=168.126.63.2 \
  --no-reverse \
  --mkhomedir \
  --unattended
```
