## 1. FPT(File Transfer Protocol)
1.1 대용량 파일 전송 서비스
1.2 Protocol: TCP
1.3 Port: 21(제어 및 인증)
1.4 Data 전송 Port:
	1.4.1 Active(능동): 20
		1.4.1.1 클라이언트가 데이터 전송 포트를 결정해서서버에 통보
		클라이언트가 알려준 포트로 서버가 데이터를 전송
		화이트보드 현상 발생: Client측 포트를 알지 못해서 방화벽에서 차단(모든 포트 해제: 이런 경우 없음, 보통 Passive 모드 사용)
	1.4.2 Passive(수동): 데이터 전송 포트 서버가 결정(관리자가 구성)
		1.4.2.1 클라이언트가 서버쪽에 데이터 전송을 요청하면 서버는 데이터 전송포트를 Client에게 통보
		클라이언트가 서버의 해당 포트에 접속해서 데이터를 전송해 감
1.5 구성
	1.5.1 `dnf install -y vsftpd`




- 도구 - 하이라이트 모음
`([0-9]{1,3})\.([0-9]{1,3})\.([0-9]{1,3})\.([0-9]{1,3})`

## 2. FTP 구성

```bash
dnf install -y dhcp-server # DHCP 설치
vi /etc/dhcp/dhcpd.conf # 환경설정 파일
------------------------------------------------------------------------------
dnf install -y vsftpd # FTP 설치
dnf autoremove -y vsftpd # FTP 삭제
rm -rf /etc/vsftpd/ # FTP 환경설정 파일 삭제
userdel -r # 계정 삭제, r 홈디렉터리, 메일 사서함 삭제 

useradd b # b 계정 생성
echo 'It1' | passwd --stdin b # b 계정 패스워드 It1로 지정

vi /etc/vsftpd/vsftpd.conf # 환경설정 파일 수정
42
52 xferlog_file=/var/log/xferlog # 주석 제거
59 idle_session_timeout=300 # 주석 제거, 300으로 변경
62 data_connection_timeout=60 # 주석 제거, 60으로 변경
86 ftpd_banner=Warning! GMSEO-FTPSERVER # 경고 문구로 수정
#banner_file=/etc/vsftpd/banner.txt # 파일 지정도 가능
101 # 주석 제거, 계정 리스트로 등록
103 # 주석 제거, vi /etc/vsftpd/chroot_list에 계정 a 추가
#chroot_list_file=/ftp/ch # 파일 경로 지정도 가능
# chroot를 적용해야 보안상으로 안전/최상위 경로인 홈디렉토리를 벗어날 수 없기 때문
128 allow_writeable_chroot=YES # 추가

systemctl enable --now vsftpd
ss -nat # n옵션 서비스명 포트번호로 변경, a tcp/udp 모두 봄, t tcp만 봄
# 서버는 통상적으로 랜카드 4개
firewall-cmd --list-all
firewall-cmd --get-zones 
firewall-cmd --get-default-zone # 기본 존 확인

firewall-cmd --permanent --zone=public --add-port=20/tcp # FTP 포트 추가
firewall-cmd --permanent --zone=public --add-port=21/tcp
# 연속된 포트 추가는 20-21 or {20,21}
firewall-cmd --permanent --remove-port= # 포트 삭제
vi /etc/firewalld/zones/public.xml # 포트 등록해둔거 삭제 가능
firewall-cmd --reload # permanent 했으면 꼭 재시작
```

```bash
#ftp 패시브모드로 사용

#FTP server 파일 생성 / 방법 2가지

dd if=/dev/zero of=/home/b/b.txt bs=300M count=1
ls -lah /home/a/a.txt #생성된 파일 확인
ls -lah /home/b/b.txt #생성된 파일 확인 / h 파일크기 보기 쉽게 변환
```
