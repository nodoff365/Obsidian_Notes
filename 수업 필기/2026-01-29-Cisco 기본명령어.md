# 라우팅 프로토콜 비교
___
### 기본 정적 라우팅
- 명령 형식
`ip route [목적지 네트워크] [서브넷마스크] [다음 홉 IP | 출구 인터페이스]`
```text
1. 다음 홉 IP 방식(권장)
ip route 192.168.3.0 255.255.255.0 10.0.0.2

이더넷 환경에서 가장 일반적
Packet Tracer 실습에서 안정적
```

```text
2. 출구 인터페이스 방식(주로 시리얼)
ip route 192.168.3.0 255.255.255.0 serial0/0/1

시리얼 링크에서 자주 사용
이더넷에서는 비권장
```
