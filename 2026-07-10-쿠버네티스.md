### 시나리오: 웹 서버 운영 중 노드 유지보수

**상황:**

- 회사 웹 서버(Nginx) 운영 중, 초기 Pod 4개로 시작
- node2 SSD 교체 필요 → drain으로 Pod 안전 이동
- 트래픽 증가로 Pod 6개로 scale up
- node1 메모리 업그레이드 필요 → cordon으로 신규 Pod 차단
- Pod 8개로 scale up (node1 제외)
- 모든 유지보수 완료 → uncordon으로 정상화
- Pod 10개로 최종 확대

**nginx-deployment.yml**
```yaml
apiVersion: apps/v1
kind: Deployment
metadata:
  name: web-server
spec:
  replicas: 4
  selector:
    matchLabels:
      app: web
  template:
    metadata:
      labels:
        app: web
    spec:
      containers:
      - name: nginx
        image: nginx
        imagePullPolicy: IfNotPresent
        ports:
        - containerPort: 80
```
