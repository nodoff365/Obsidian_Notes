## Import/Export
```bash
#rocky9-1
# 1) 컨테이너 실행 (httpd + index 수정용)
docker run -dit --name web1 httpd

# 2) index.html 수정
docker exec web1 sh -c 'echo "rocky9-1" > /usr/local/apache2/htdocs/index.html'

# 3) export (컨테이너 → tar 파일)
docker export -o httpd.tar web1

# 4) 파일 전송
scp httpd.tar root@10.0.0.12:/root/
```

```bash
#rocky9-2
# 1) 기존 이름/이미지 충돌 방지 (있으면 삭제)
docker rm -f h1 2>/dev/null
docker rmi -f httpd:1.0 2>/dev/null

# 2) import (CMD 꼭 추가해야 실행됨)
docker import \
--change 'CMD ["httpd-foreground"]' \
--change 'ENV PATH=/usr/local/apache2/bin:/usr/local/sbin:/usr/local/bin:/usr/sbin:/usr/bin:/sbin:/bin' \
httpd.tar httpd:1.0

# 3) 실행 (포트 60080 + 컨테이너 이름 h1)
docker run -d --name h1 -p 60080:80 httpd:1.0

# 4) 확인 (컨테이너 상태)
docker ps

# 5) 웹 확인
curl localhost:60080

# 6) index 확인 (Rocky9-1에서 수정된 내용 유지 확인)
docker exec h1 cat /usr/local/apache2/htdocs/index.html
```
