# ACL

1)Standard ACL

| 구분                  |      번호 범위      |
| ------------------- | :-------------: |
| Standard ACL        |   **1 ~ 99**    |
| 확장 Standard (추가 범위) | **1300 ~ 1999** |

2)Extended ACL

| 구분                  |      번호 범위      |
| ------------------- | :-------------: |
| Extended ACL        |  **100 ~ 199**  |
| 확장 Extended (추가 범위) | **2000 ~ 2699** |

### 스위치 기본 설정
```python
#Sw-net1 switch
enable
configure terminal

hostname Sw-net1              # 장비 이름 설정

enable password boan          # enable 평문 비밀번호
enable secret class           # enable 암호화 비밀번호 (우선 적용)

no ip domain-lookup           # 오타 입력 시 DNS 조회 방지

username boan secret class    # 로컬 사용자 계정 생성

line vty 0 15                 # 원격 접속 라인 설정
login local                  # 로컬 계정 인증 사용
exit

interface vlan1               # 관리용 SVI 인터페이스
ip address 192.168.1.2 255.255.255.0  # 관리 IP 설정
no shutdown
exit

ip default-gateway 192.168.1.1  # L2 스위치 기본 게이트웨이

service password-encryption   # 평문 패스워드 암호화

banner motd # Local account(boan/class), enable(class) #

end
copy running-config startup-config   # 설정 저장

#Sw-net2 switch
enable
configure terminal
hostname Sw-net2
enable password boan
enable secret secret
no ip domain-lookup
username boan secret secret
line vty 0 15
login local
exit
interface vlan1
ip address 192.168.2.2 255.255.255.0
no shutdown
ip default-gateway 192.168.2.1
service password-encryption
banner motd # Local account(boan/secret), enable(secret) #
end
copy running-config startup-config
```

```python
#net1 Router
enable
configure terminal
hostname net1

interface GigabitEthernet0/0
ip address 192.168.1.1 255.255.255.0   # 내부망 게이트웨이
no shutdown
exit

interface Serial0/0/0
ip address 201.21.21.2 255.255.255.252 # WAN 연결
no shutdown
clock rate 125000                     # DCE일 경우 설정
exit

ip route 0.0.0.0 0.0.0.0 201.21.21.1    # 기본 경로

end
copy running-config startup-config

#net2 Router
enable
configure terminal
hostname net2

no shutdown
exit

ip address 201.21.21.1 255.255.255.252
no shutdown
exit

ip route 0.0.0.0 0.0.0.0 201.21.21.2

end
copy running-config startup-config
```
