#### 2.1 Rhel OS 설정
> RHEL 설치 후 서버 운영에 필요한 기본 OS 설정을 진행한다.  
이 단계에서는 **hostname 지정, IP 주소 설정, 네트워크 구성 등 시스템 초기 설정을 수행한다.**

![[Pasted image 20260314114030.png]]
1. ㅁㄴㅇㄹ

- Clone 하기전 반드시 해야하는 초기화 작업
```bash
#!/bin/bash
set -euo pipefail
  
echo "[INFO] Template cleanup start"
  
# 1) machine-id 초기화
truncate -s 0 /etc/machine-id
rm -f /var/lib/dbus/machine-id
  
# 2) SSH host key 삭제
rm -f /etc/ssh/ssh_host_*
  
# 3) NetworkManager 연결 프로파일 삭제
rm -f /etc/NetworkManager/system-connections/*
  
# 4) hostname 초기화
hostnamectl set-hostname localhost.localdomain
  
cat > /etc/hosts <<'EOF'
127.0.0.1   localhost localhost.localdomain
::1         localhost localhost.localdomain
EOF
  
# 5) DNF 캐시 정리
dnf clean all || true
  
# 6) journal 로그 축소
journalctl --rotate || true
journalctl --vacuum-time=1s || true
  
# 7) 임시 파일 정리
rm -rf /tmp/*
rm -rf /var/tmp/*
  
# 8) root history 정리
history -c || true
rm -f /root/.bash_history
  
# 9) cloud-init 사용 시 초기화
if command -v cloud-init >/dev/null 2>&1; then
    cloud-init clean || true
    rm -rf /var/lib/cloud/*
fi
  
# 10) httpd 로그 구조 보장
mkdir -p /var/log/httpd
if [[ ! -L /etc/httpd/logs ]]; then
    rm -rf /etc/httpd/logs
    ln -s ../../var/log/httpd /etc/httpd/logs
fi
  
# 11) SELinux 컨텍스트 복구
restorecon -Rv /etc/ssh /etc/httpd /var/log >/dev/null 2>&1 || true
  
echo "[INFO] Template cleanup completed"
echo "[INFO] Power off this VM, then create snapshot/full clone"
```

- Clone 방법
```text
복제할 VM 우클릭 > Manage > Clone > 다음 > The current state in the virtual machine > Create a full clone > VM 이름, 저장 위치 지정 후 마침
```

- Clone 후 설정
```bash
sudo systemd-machine-id-setup
sudo ssh-keygen -A
timedatectl
chronyc sources -V
sudo hostnamectl set-hostname [호스트명]
```
