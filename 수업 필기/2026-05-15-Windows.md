## 그룹정책
#### ADMX 중앙 저장소 (Central Store)

**개념:** GPO 관리 템플릿 파일을 SYSVOL에 올려서 모든 관리자가 동일한 정책 화면을 보도록 통일하는 것

---

**실습 순서**

```
C:\Windows\PolicyDefinitions  (로컬 ADMX 파일 위치)
        ↓ 복사
\\sgm.local\SYSVOL\sgm.local\Policies\PolicyDefinitions  (중앙 저장소)
```

---

**적용 확인**

```
gpmc.msc → GPO 편집
→ 컴퓨터 구성 또는 사용자 구성 → 정책 → 관리 템플릿
```

|적용 전|적용 후|
|---|---|
|로컬 컴퓨터의 정책 정의(ADMX 파일)를 검색했습니다|중앙 저장소에서 정책 정의를 검색했습니다|

---

**효과**

|로컬 저장소|중앙 저장소|
|---|---|
|PC마다 버전 다를 수 있음|도메인 전체 동일 버전|
|관리자마다 화면 다를 수 있음|모든 관리자 동일 화면|
|Edge/Chrome 정책 추가 불편|ADMX 파일 한번만 추가하면 전체 적용|

> 현업 AD에서는 여러 관리자가 GPO를 다루는 환경에서 중앙 저장소를 주로 사용

**관리 템플릿** 항목을 확인
```
GPO 편집 → 컴퓨터 구성 또는 사용자 구성
→ 정책 → 관리 템플릿
```

중앙 저장소 적용 **전:**
```
관리 템플릿: 로컬 컴퓨터의 정책 정의(ADMX 파일)를 검색했습니다.
```

중앙 저장소 적용 **후:**
```
관리 템플릿: 중앙 저장소에서 정책 정의를 검색했습니다.
```
