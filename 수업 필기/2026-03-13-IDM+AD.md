## IDM(FreeIPA)+Active Directory 통합 인프라 구축
---
### 네트워크 및 서버 구성
![[이미지.png]]
```
네트워크 및 서버 구성도

kcci.edu (FreeIPA DNS, 10.10.20.10)  ← authoritative zone / kcci.edu 도메인 DNS 관리
│
├ Linux Servers                      ← FreeIPA DNS가 직접 관리하는 호스트 영역
│   │
│   ├ idm.kcci.edu                   ← FreeIPA 서버 (IdM + DNS + CA)
│   │
│   ├ web.kcci.edu                   ← Apache + ModSecurity
│   │                                  외부 HTTPS 처리 및 내부 서비스 프록시
│   │
│   └ bastion.kcci.edu               ← Jump/Bastion Server
│                                      운영자가 내부 서버 접속 시 사용하는 중계 서버
│
└ ad.kcci.edu (Delegation)           ← FreeIPA DNS에서 AD DNS로 위임된 하위 도메인
      │
      ├ ads.ad.kcci.edu              ← AD DNS / Domain Controller (10.10.20.15)
      │                                ad.kcci.edu 영역 authoritative DNS
      │                                Active Directory 인증 및 도메인 관리
      │
      └ pc1.ad.kcci.edu              ← Windows Client (10.10.20.16)
                                       AD 도메인에 가입된 사용자 PC
```
